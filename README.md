# 打地鼠游戏

一个简单有趣的打地鼠游戏，完全适配移动端，可在iOS Safari上流畅运行。

## 特点

- 响应式设计，适配各种屏幕尺寸
- 触摸屏优化，支持移动端操作
- 精美的动画效果
- 计分系统
- 倒计时功能

## 游戏玩法

1. 点击"开始游戏"按钮
2. 等待3秒倒计时
3. 当地鼠出现时点击它们
4. 30秒内尽可能获得更高的分数

## 技术栈

- HTML5
- CSS3
- JavaScript
